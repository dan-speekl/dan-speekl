### 🔭 What I'm working on

**[Traque](https://github.com/traque-dev)** — a modern error and event tracking platform for startups.

| Repo | What it does |
| --- | --- |
| [traque-backend](https://github.com/traque-dev/traque-backend) | Error and event tracking service built with **NestJS** — monitor, track, and analyze in real time. |
| [traque-js](https://github.com/traque-dev/traque-js) | JS/TS SDKs for error and event capture across web, Node, React, and Nest. |
| [traque-frontend](https://github.com/traque-dev/traque-frontend) | Web client built with **React, Vite, TanStack Router/Query, and Tailwind CSS**. |
| [traque.dev](https://github.com/traque-dev/traque.dev) | **Next.js** docs and marketing site with SDK guides. |

**Other projects**

- [**better-activity**](https://github.com/better-activity/better-activity) — Framework-agnostic, multi-database audit logging modeled on the architecture of `better-auth`. Declare your entities once, log everything, query it with confidence.
- [**i18n-email**](https://github.com/i18n-email/i18n-email) — Translate transactional emails into any language using AI. Works with React Email and raw HTML.
